**Adding Permissions**

GreedyGame SDK needs the following permissions to work with.

**Mandatory permissions**

 * Add `App Transport Security Settings` in your project plist file.
 * Then make `Allow Arbitary Loads` to `YES`.

  <center><img src="/img/newiOS/Internet-Permission.png" alt="" style="margin-left: 0px" width="700" height="36"></center>


**Optional permissions**     

 Add any one of the location permission depends upon your requirement.We prefer to set `Privacy-Location When In Use Usage Description` in plist file.

   <center><img src="/img/newiOS/Location-permission.png" alt="" style="margin-left: 0px" width="700" height="36"></center>


!!! tip
    Location permission will help improving the revenue because of better ad targeting.
